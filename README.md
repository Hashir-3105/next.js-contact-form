# Contact form Project

## Features
- Form fields: First Name, Last Name, Email, Query Type, Message, Consent
- Validation with React Hook Form + Zod
- Success notification via Sonner Toast
- Mobile-first responsive layout using Tailwind CSS
- Custom reusable components: Input, Textarea, Radio, Checkbox
- Accessible: screen-reader-friendly
- Default route `/` redirects to `/contact-us`
- Dynamic theme support, no hardcoded colors

## Tech Stack
Next.js 15 | React | Tailwind CSS | Shadcn/ui | React Hook Form | Zod | Sonner

Responsive grid layout for all screen sizes
